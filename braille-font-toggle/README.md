# Braille Font Toggle (Chrome Extension, MV3)

This folder contains the Chrome extension described in the repo root README.

## Load unpacked
1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this folder: `braille-font-toggle/`

## Important
- Replace `fonts/BrailleFont.woff2` with a real Braille font file you have rights to use.
- Click the extension toolbar button to toggle the Braille override for the active tab.
- When enabled, it automatically reapplies on navigation and updates dynamically for new content (useful for SPAs/infinite-scroll pages).
